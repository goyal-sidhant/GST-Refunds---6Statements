Using a Free HEX Editor to Remove a VBA Password

Follow these step-by-step instructions:

1. Close the Excel File: Ensure the Excel file with the locked VBA project is closed.

2. Change File Extension: Go to the folder containing the Excel file and change its extension from .xlsm to .zip. Confirm the change when prompted.

3. Access the Zip Folder: Double-click the newly created .zip folder to open it.

4. Extract VBA project.bin: Navigate into the Excel folder inside the zip file and copy the VBA project.bin file out to your working directory.

5. Open VBA project.bin in Hex Editor: Use a hex editor, such as HxD Editor, to open the VBA project.bin file.

6. Search for dpb: In the hex editor, use the search function (Ctrl + F) to find the string dpb. This will lead you to a sequence starting with dpb= followed by alphanumeric characters in double quotes.

7. Copy the Password String: Copy the alphanumeric characters (excluding dpb=) that follow dpb= and paste them into a notepad.

8. Replace the String: In the hex editor, carefully select the characters you just copied (which represent the unknown password) and replace them with the following sequence that corresponds to the password "123":
   
   0A08A6B1B6CEB6CE4932B7CE4B63A66D37B84BA3D4BAD58A6B495254585A5D3D675777675D
   
   Ensure the length of the new sequence matches the old one, padding with zeros if necessary.

9. Save Changes: Save the modified VBA project.bin file in the hex editor.

10. Replace VBA project.bin in Zip: Go back to the original .zip folder, navigate to the Excel folder, and replace the old VBA project.bin file with your newly modified one.

11. Change Extension Back: Change the .zip file extension back to .xlsm.

12. Open File and Enter New Password: Open the Excel file. When prompted for the VBA password, enter "123" (which is the password you set using the hex editor) to unlock the VBA project.

13. Optional: Remove Password Permanently: Once unlocked, you can go into the VBA project properties and remove the password entirely by unchecking the "Lock project for viewing" option.
