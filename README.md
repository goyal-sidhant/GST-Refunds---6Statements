# GST Refund JSON Generator — 6 Statements

**What it does:** Reads filled-in Excel templates for GST Refund statements and produces portal-ready JSON files that can be uploaded directly to the GST Portal's offline utility.

**Built by:** Adv. Sidhant Goyal, Goyal Tax Services Pvt. Ltd.
**Last updated:** 12-03-2026
**Version:** 1.0
**Status:** In Progress — all 6 statements built, GUI integration testing pending

---

## How to Use

### Step 1: Open the Tool
Open terminal in this folder and run `python main.py`

### Step 2: Select Statement Type
Choose the refund statement type from the dropdown (S01A, S02, S03, S04, S05, or S06).

### Step 3: Generate Template
Click **Generate Template**. A formatted Excel file with correct headers, column widths, and dropdown validations will be saved to your chosen location.

### Step 4: Fill in the Template
Open the generated Excel file. Enter details (GSTIN, financial year, tax period, invoice data, etc.) in the appropriate rows. Follow the column headers and dropdown options exactly.

### Step 5: Load the Filled Template
Drag-and-drop the filled Excel template into the app, or use the file selector.

### Step 6: Review Validation Results
The app validates all data — GSTINs, tax amounts, dates, duplicates, BRC/FIRC linkages, and 50+ business rules. Errors and warnings appear in the log panel.

### Step 7: Generate JSON
If validation passes, click **Generate JSON**. The portal-ready `.json` file is saved to the output folder.

## Common Problems and Solutions

### "File may be open in Excel"
**Why:** The Excel file is locked by another program.
**Fix:** Close the file in Excel, then try again.

### GSTIN validation errors on every row
**Why:** Extra spaces, lowercase letters, or copy-paste formatting in the GSTIN column.
**Fix:** In Excel, select the GSTIN column → Find & Replace → find a space, replace with nothing. Ensure all letters are uppercase.

### "Template structure mismatch"
**Why:** The Excel file was not generated by this tool, or sheet names / column headers were manually edited.
**Fix:** Generate a fresh template from the app, then copy your data into it without changing headers or sheet names.

### BRC/FIRC linkage errors (S02, S03 only)
**Why:** BRC/FIRC rows are not properly linked to their invoice rows.
**Fix:** Ensure BRC/FIRC data is in the same row as the invoice (adjacent mode), or use the Group ID column to link them explicitly.

### Tax amount mismatch warning
**Why:** Document Value does not equal Taxable Value + IGST + CGST + SGST + Cess.
**Fix:** Check and correct the amounts so they add up.

## What This Tool Does NOT Do

- Does not file the return — only prepares the JSON upload file.
- Does not connect to the internet — works fully offline.
- Does not validate against the GST Portal — structural validation only.
- Does not auto-update — if GST rules or JSON schema change, the tool needs manual update.

## Setup (for a new computer)

### Requirements
- Windows 10 or 11
- Python 3.10 or higher

1. Copy the entire project folder to your computer.
2. Open terminal (Command Prompt) in the folder.
3. Run: `pip install PyQt5 openpyxl`
4. Run: `python main.py` to start the tool.
