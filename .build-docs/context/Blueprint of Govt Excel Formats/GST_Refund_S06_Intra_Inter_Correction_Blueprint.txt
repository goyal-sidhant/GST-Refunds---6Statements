================================================================================
2. PURPOSE & LEGAL BASIS
================================================================================

This template covers Section 77 / Section 19 situations where tax was paid
treating a supply as intra-State (or inter-State), but later it is held to be
the opposite due to correction/change in Place of Supply.

Each row is a COMPARISON ROW:
  "How it was taxed earlier" vs. "How it should be taxed now"

This is the WIDEST template (22 columns, A through V) and is:
  - Evidence-heavy and classification-heavy
  - NOT export-evidence-heavy (no Port Code / Shipping Bill / BRC)
  - Unique in requiring B2B vs B2C recipient identification
  - Unique in having "Earlier" vs "Correct" paired blocks

REFUND CATEGORY: Correction/Reclassification — neither purely tax-paid nor
purely ITC-based. The refund arises from the difference between tax paid
under wrong classification and tax correctly payable.

--- RECIPIENT IDENTIFICATION BLOCK (Columns A–D) ---

Col | Field                  | Notes
----|------------------------|----------------------------------------------
A   | Sr. No.                | Positive integer

--- INVOICE BLOCK (Columns E–I) ---

Col | Field                  | Notes
----|------------------------|----------------------------------------------
E   | Type of Document       | Invoice / Debit Note / Credit Note
F   | Document No.           | Max 16 alphanumeric + /-
G   | Document Date          | dd-mm-yyyy; standard date validations
H   | Document Value         | Numeric, >= 0
I   | Taxable Value          | Numeric, >= 0

--- EARLIER CLASSIFICATION BLOCK (Columns J–O) ---
     "How the transaction was originally treated"

Col | Field                      | Notes
----|----------------------------|------------------------------------------
J   | Earlier Classification     | Inter / Intra
    |                            |   Drives K vs L-M lock/unlock (see Sec 6)
K   | Earlier IGST               | Enabled if Earlier = Inter; locked if Intra
L   | Earlier CGST               | Enabled if Earlier = Intra; locked if Inter
M   | Earlier SGST/UTGST         | Same logic as CGST
N   | Earlier Cess               | Optional
O   | Earlier Place of Supply    | State/UT code from dropdown master list

--- CORRECT CLASSIFICATION BLOCK (Columns P–U) ---
     "How the same transaction should now be treated"

Col | Field                      | Notes
----|----------------------------|------------------------------------------
P   | Correct Classification     | Inter / Intra (must differ from Col J)
Q   | Correct IGST               | Enabled if Correct = Inter
R   | Correct CGST               | Enabled if Correct = Intra
S   | Correct SGST/UTGST         | Same logic as CGST
T   | Correct Cess               | Optional
U   | Correct Place of Supply    | State/UT code from dropdown master list

--- ERROR COLUMN ---

Col | Field                      | Notes
----|----------------------------|------------------------------------------
V   | Error                      | Auto-populated by VBA — do NOT type manually

================================================================================
5. B2B vs B2C LOCK/UNLOCK (VBA BEHAVIOUR)
================================================================================

WHEN "B2C" IS SELECTED (Column B):
  - Col C (GSTIN/UIN) is LOCKED.
    Display: "Disabled" in grey (ColorIndex = 15, Font.ColorIndex = 15).
  - Col D (Name) is UNLOCKED for free-text entry.
    Display: white background, black font.

WHEN "B2B" IS SELECTED (Column B):
  - Col D (Name) is LOCKED.
    Display: "Disabled" in grey.
  - Col C (GSTIN/UIN) is UNLOCKED for GSTIN entry.
    Display: white background, black font.

WHEN "CLEAR" IS SELECTED (Column B):
  - All running total cells in row 8 (K through U) are CLEARED and RESET.
  - Background reset to accent colour (ColorIndex = 17).

================================================================================
6. INTER/INTRA TAX LOCK LOGIC
================================================================================

This logic applies INDEPENDENTLY to BOTH the Earlier block and the Correct block:

WHEN "INTER" IS SELECTED:
  - IGST field: ENABLED (editable)
  - CGST field: LOCKED (disabled)
  - SGST field: LOCKED (disabled)

WHEN "INTRA" IS SELECTED:
  - IGST field: LOCKED (disabled)
  - CGST field: ENABLED (editable)
  - SGST field: ENABLED (editable)

The VBA dynamically enables/disables the tax columns based on Inter/Intra
selection for both the "before" (J) and "after" (P) classification blocks.

IMPORTANT: The Earlier and Correct classifications must be OPPOSITE to each
other — if Earlier is Inter, Correct must be Intra, and vice versa. This is
the whole basis of the refund claim.

================================================================================
7. VALIDATION RULES
================================================================================

RECIPIENT:
  - If B2B: GSTIN/UIN must be valid and present.
  - If B2C: Name must be provided.

DOCUMENT:
  - Document type, number, date, and value are mandatory.
  - Standard date validations (>= 01-Jul-2017, <= current, leap year aware).

TAX AMOUNTS:
  - Earlier block: If Inter → only IGST; if Intra → only CGST+SGST.
  - Correct block: Same mutual exclusivity rules.
  - Cess is optional in both blocks.
  - All tax amounts must be numeric and >= 0.

PLACE OF SUPPLY:
  - Must be selected from the State/UT master list dropdown.

DUPLICATE DETECTION:
  Standard duplicate check on document references.

PER-ROW JSON KEYS:

JSON Key | Source Col  | Meaning
---------|------------|------------------------------------------
sno      | A (Col 1)  | Serial number
invType  | B (Col 2)  | B2B or B2C
uin      | C (Col 3)  | Recipient GSTIN/UIN (or blank for B2C)
docType  | E (Col 5)  | Document type
inum     | F (Col 6)  | Document number
idt      | G (Col 7)  | Document date
val      | H (Col 8)  | Document value
txVal    | I (Col 9)  | Taxable value
bInt     | J (Col 10) | Earlier classification (Inter/Intra)
bCGST    | L (Col 12) | Earlier CGST
bSGST    | M (Col 13) | Earlier SGST
bCess    | N (Col 14) | Earlier Cess (OMITTED if blank)
bPOS     | O (Col 15) | Earlier Place of Supply
aInt     | P (Col 16) | Correct classification (Inter/Intra)
aIGST    | Q (Col 17) | Correct IGST
aCess    | T (Col 20) | Correct Cess (OMITTED if blank)
aPOS     | U (Col 21) | Correct Place of Supply

JSON CONDITIONAL LOGIC — FOUR PATHS:
  Path 1: Both bCess (Col 14) and aCess (Col 20) present → all keys emitted.
  Path 2: Both bCess and aCess blank → bCess and aCess omitted.
  Path 3: bCess blank, aCess present → bCess omitted, aCess included.
  Path 4: bCess present, aCess blank → bCess included, aCess omitted.

NOTE ON VBA CODE STRUCTURE:
  The S06 createJsonFile function has extensive commented-out conditional paths
  (dozens of nested If/Else blocks for every combination of blank tax fields).
  The active code uses the four-path approach based solely on bCess/aCess
  presence. The JSON is built through helper functions (B2BInter, B2BIntra,
  B2CInter, B2CIntra, B2BIntraUIN) that are selected based on the combination
  of invType (B2B/B2C) and classification (Inter/Intra).

================================================================================
11. UNIQUE CHARACTERISTICS OF S06
================================================================================

1. WIDEST FORMAT: 22 columns (A through V) vs 9 for S05 (the lightest).

2. PAIRED BLOCKS: Only statement with "Earlier" vs "Correct" comparison in
   a single row — every other statement has a single tax/evidence block.

3. B2B/B2C TOGGLE: Only statement where recipient identification changes
   dynamically based on B2B vs B2C selection (GSTIN vs Name).

4. INTER/INTRA TOGGLE: Only statement with dynamic tax column lock/unlock
   based on Inter vs Intra classification — and it happens TWICE per row
   (once for Earlier, once for Correct).

5. PLACE OF SUPPLY: Only statement with explicit State/UT code POS fields
   (both Earlier and Correct).

6. NO EXPORT EVIDENCE: No Port Code, Shipping Bill, EGM, BRC/FIRC — this
   is entirely about domestic classification correction.

APPENDIX A: GST PORTAL TUTORIAL LINK
================================================================================

Official Tutorial URL:
https://tutorial.gst.gov.in/userguide/refund/File_reply_manual.htm

Portal Workflow (summarised from tutorial.gst.gov.in):
  1. Login to GST Portal → Services → Refunds → Application for Refund
  2. Select the appropriate refund type from the dropdown
  3. Select the Tax Period and click CREATE REFUND APPLICATION
  4. IMPORTANT: All applicable returns (GSTR-1, GSTR-3B) must be filed
     for the period before refund application can be created
  5. Download the Offline Utility (.xlsb file) from the portal
  6. Fill in the data in the Excel template
  7. Click 'Validate & Calculate' to check for errors
  8. Rectify any errors shown in the Error column
  9. Click 'Create File To Upload' to generate the JSON file
  10. Upload the JSON file on the GST Portal
  11. Click 'Validate Statement' on the portal
  12. If invalid documents exist, download error sheet, fix, re-upload
  13. Once validated, proceed to fill refund computation table
  14. Select bank account, upload supporting documents
  15. Save the application (valid for 15 days)
  16. File with DSC or EVC
  17. ARN is generated and sent via email/SMS

KEY PORTAL NOTES:
  - System does NOT validate period of limitation (Section 54(1)).
    Taxpayer must ensure filing within 2 years from relevant date.
  - Same invoice cannot be reused in another refund application.
  - If deficiency memo is issued or application withdrawn, invoices
    become available for reuse.
  - Up to 15 supporting documents (5 MB each) can be uploaded.
  - Nil refund can be filed if no refund is to be claimed for a period.

Function getDataFromSheet()
    Dim currentWS As Worksheet
    Set currentWS = ActiveSheet
    Dim lastCell As Range
    
    
    With Sheets("RFD_STMT06")
        Set lastCell = Range("A13:U10012").Find(What:="*", _
        After:=.Range("A13"), _
        Lookat:=xlPart, _
        LookIn:=xlFormulas, _
        SearchOrder:=xlByRows, _
        SearchDirection:=xlPrevious, _
        MatchCase:=False)
    End With
    
    If lastCell Is Nothing Then
        LASTROW = 12
    Else
        LASTROW = lastCell.Row
    End If
    
    If lastCell Is Nothing Then
        STARTROW = 0
        ENDROW = 0
        WS_DATA = Null
    Else
        WS_DATA = currentWS.Range("A13:V" & LASTROW).Value
        STARTROW = LBound(WS_DATA, 1)
        ENDROW = UBound(WS_DATA, 1)
    End If
    
    Set currentWS = Nothing
End Function

Function writeDataToSheet()
    
    Range("A13:V" & LASTROW).Value = WS_DATA

'Select the cells which are below the last row containing value in coulmn A. Clear those selcted cells.
    Range(Cells(LASTROW + 1, "V"), Cells(LASTROW + 1, "V").End(xlDown)).ClearContents
    
    
End Function

Sub autoGenSerial()

    Dim serialNo As Integer
    Dim Row As Integer
    
    serialNo = 1
        
    For Row = STARTROW To ENDROW
        'Debug.Print WS_DATA(Row, 2)
        
        If WS_DATA(Row, 2) <> "" Then
           WS_DATA(Row, 1) = serialNo
           serialNo = serialNo + 1
        End If

End Sub

Function checkAndCreateJson()
    Dim errorCount, addCount, rowNo As Integer
    Dim errMsg As String
    
    errMsg = " Exists in the sheet. Please click Create File To Upload after resolving the errors"
    
    addCount = 0
    
    If Not (STARTROW = 0 And ENDROW = 0) Then
        For rowNo = STARTROW To ENDROW Step 1
            
            If Not WS_DATA(rowNo, 22) = "" Then
                errorCount = errorCount + 1
            End If
            
        Next
    End If
    
    If (STARTROW = 0 And ENDROW = 0) Then
        
        MsgBox "Please type atleast one record to create file to upload", vbCritical
        
    ElseIf errorCount = 0 Then
        
        If Cells(5, "C") <> "" Then
            MsgBox "Valid GSTIN is Mandatory", vbCritical
        ElseIf Cells(6, "J") <> "" Then
            MsgBox "Valid Order Number is Mandatory", vbCritical
        ElseIf Cells(6, "M") <> "" Then
            MsgBox "Valid Order Date is Mandatory", vbCritical
        Else
            autoGenSerial
            createJsonFile
        End If
        
    ElseIf errorCount = 1 Then
        MsgBox errorCount & " error" & errMsg, vbCritical
    Else
        MsgBox errorCount & " errors" & errMsg, vbCritical
    End If
    
End Function

Function createJsonFile()
    Dim rowNo As Integer
    Dim path, json As String
    Dim fs As Object, jsonfile
    checkUIN = False
    checkGSTIN = False
    
    path = Application.GetSaveAsFilename
    
    If path <> "False" Then
        
        Set fs = CreateObject("Scripting.FileSystemObject")
        
        path = path + FILE_TYPE
        Set jsonfile = fs.CreateTextFile(path, True)
        
        jsonfile.Write ("{""gstin"":""")
        jsonfile.Write (Cells(5, "B"))
        jsonfile.Write (""",""refundRsn"":""")
        jsonfile.Write ("INTRVC")
        jsonfile.Write (""",""version"":""")
        jsonfile.Write ("2.0")
        jsonfile.Write (""",""OdrNo"":""")
        jsonfile.Write (Cells(6, "I"))
        jsonfile.Write (""",""OdrDt"":""")
        jsonfile.Write (Cells(6, "L"))
        jsonfile.Write (""",""stmt06"":[{")
        
        For rowNo = STARTROW To ENDROW Step 1
        
         If Not WS_DATA(rowNo, 3) = "" Then
                If Not WS_DATA(rowNo, 3) = "Disabled" Then
                    OBJ_RE.Pattern = GSTIN_REGEX
                    If Not OBJ_RE.Test(WS_DATA(rowNo, 3)) Then
                        OBJ_RE.Pattern = UIN_REGEX
                        If Not OBJ_RE.Test(WS_DATA(rowNo, 3)) Then
                            WS_DATA(rowNo, 22) = "Please Enter Valid GSTIN/UIN of Recipient"
                        Else
                            checkUIN = True
                        End If
                    Else
                        checkGSTIN = True
                    End If
                End If
            End If
            
            If (WS_DATA(rowNo, 2)) = "B2C" And (WS_DATA(rowNo, 10)) = "Inter" Then
                
                json = B2CInter(json, rowNo)
                
            Else
                
                If (WS_DATA(rowNo, 2)) = "B2C" And (WS_DATA(rowNo, 10)) = "Intra" Then
                    
                    json = B2CIntra(json, rowNo)
                    
                Else
                    
                    If (WS_DATA(rowNo, 2)) = "B2B" And (WS_DATA(rowNo, 10)) = "Inter" And checkGSTIN = True Then
                        
                        json = B2BInterGSTIN(json, rowNo)
                        
                    Else
                        
                        If (WS_DATA(rowNo, 2)) = "B2B" And (WS_DATA(rowNo, 10)) = "Inter" And checkUIN = True Then
                            
                            json = B2BInterUIN(json, rowNo)
                            
                        Else
                            
                            If (WS_DATA(rowNo, 2)) = "B2B" And (WS_DATA(rowNo, 10)) = "Intra" And checkGSTIN = True Then
                                
                                json = B2BIntraGSTIN(json, rowNo)
                                
                            Else
                                
                                If (WS_DATA(rowNo, 2)) = "B2B" And (WS_DATA(rowNo, 10)) = "Intra" And checkUIN = True Then
                                    
                                    json = B2BIntraUIN(json, rowNo)
                                    
                                End If
                            End If
                        End If
                    End If
                End If
            End If
            
            If rowNo = ENDROW Then
                json = json & ("""}")
            Else
                json = json & ("""},{")
            End If
            
            jsonfile.Write json
            
        Next
        
        jsonfile.Write ("]}")
        
        jsonfile.Close
        
        MsgBox "File has been created and placed in  " & path & "  ! You may now proceed to upload the File in the GST Web Portal", vbInformation
        
        Set fs = Nothing
        Set jsonfile = Nothing
        
    End If
    
End Function
